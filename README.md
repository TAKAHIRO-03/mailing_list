



Name
====
vbaでNotesメール一斉送信＆名簿管理

## Overview
VBAのユーザーフォームを使用して、Notesと連携し、Sheetに入ってある情報に基づき、一斉送信。

オリジナルのメーリングリストを作成したい方はぜひ。

Excel2016で動作確認済み
